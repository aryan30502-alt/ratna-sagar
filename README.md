## Features
- ✅ Premium luxury catalogue website
- ✅ Live Gold & Silver rate ticker (auto-sync + manual override)
- ✅ 7 jewellery collection categories
- ✅ WhatsApp enquiry integration
- ✅ Admin panel (rates, products, enquiries, collections)
- ✅ Mobile responsive with native-app-feel bottom bar
- ✅ BIS Hallmark & GIA trust signals
- ✅ SEO optimized

## Tech Stack
- Next.js 14 (Static Export)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- Deployed on Netlify
